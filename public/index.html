cURL Example

curl -X POST http://localhost:5000/api/v1/lightning/get_speech \ -H "Authorization: Bearer YOUR_API_TOKEN" \ -H "Content-Type: application/json" \ -d '{ "voice_id": "en-US-GuyNeural", "text": "Hello from Enterprise Voice TTS", "sample_rate": 24000, "add_wav_header": true }'

JavaScript (Fetch API)

const options = { method: 'POST', headers: { 'Authorization': 'Bearer YOUR_API_TOKEN', 'Content-Type': 'application/json' }, body: JSON.stringify({ voice_id: 'en-US-GuyNeural', text: 'Hello from Enterprise Voice TTS', sample_rate: 24000, add_wav_header: true }) }; fetch('http://localhost:5000/api/v1/lightning/get_speech', options) .then(response => response.json()) .then(data => console.log(data)) .catch(err => console.error(err));

Python (requests)

import requests url = "http://localhost:5000/api/v1/lightning/get_speech" headers = { "Authorization": "Bearer YOUR_API_TOKEN", "Content-Type": "application/json" } data = { "voice_id": "en-US-GuyNeural", "text": "Hello from Enterprise Voice TTS", "sample_rate": 24000, "add_wav_header": True } response = requests.post(url, headers=headers, json=data) print(response.json())

Node.js (axios)

const axios = require('axios'); const response = await axios.post( 'http://localhost:5000/api/v1/lightning/get_speech', { voice_id: 'en-US-GuyNeural', text: 'Hello from Enterprise Voice TTS', sample_rate: 24000, add_wav_header: true }, { headers: { 'Authorization': 'Bearer YOUR_API_TOKEN', 'Content-Type': 'application/json' } } ); console.log(response.data);
